## 🏗️ SALES PAGE DECONSTRUCTOR ANALYST (Detailed)

### Core Capabilities

**1. Structure Extraction**
- Identify sections (headline, lead, mechanism, proof, offer, CTA)
- Map flow sequence
- Measure section lengths, transitions
- Extract CTA positioning

**2. Mechanism Analysis**
- Extract promise (primary, emotional, tangible, timeframe)
- Identify mechanism (name, category, explanation, unique element)
- Map vs. alternatives (why they fail)
- Extract belief shifts (from → to)
- List proof pillars (type, strength)

**3. Offer Deconstruction**
- Extract value equation (Dream Outcome, Likelihood, Time Delay, Effort)
- Map value stack (core + bonuses)
- Analyze pricing psychology (anchoring, scarcity, guarantee)
- Document CTA structure

**4. Voice Signature**
- Analyze tone markers (formality, warmth, authority, urgency)
- Measure sentence rhythm (short/medium/long ratio)
- Extract word choice patterns (second-person ratio, active voice, sensory words)
- Identify forbidden words avoided
- Catalog power phrases used

**5. 7S/7F Coverage Audit**
- Score each dimension (SAFE, SPECIAL, SMART, SIGNIFICANT, SUPPORTED, SUPERIOR, STEAL)
- Identify gaps
- Assess balance
- Map to Neuro-Box 6D axes

**6. Pattern Recognition**
- Match against Pattern Library
- Flag new patterns discovered
- Confirm existing patterns
- Identify anti-patterns avoided

**7. Patch Proposal Generation**
- Identify skill gaps
- Propose specific improvements
- Prioritize patches (HIGH/MEDIUM/LOW)
- Estimate implementation effort
