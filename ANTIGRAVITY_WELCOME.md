## 🧠 Progressive Disclosure (How Context Stays Light)

Instead of loading all 30 skills at once, the system loads expertise **on-demand** in layers:

```
L1: Quick Reference    (400-600 tokens)   ← "What is this skill?"
L2: Core Procedure     (1,500-4,000 tokens) ← "How do I execute?"
L3: Advanced Patterns  (1,000-3,000 tokens) ← "Edge cases?"
L4: Output Templates   (600-2,000 tokens)   ← "What does output look like?"
```

**Example Flow:**
```
You: "/advertorial"
   ↓
System loads: advertorial_master L1 (600 tokens)
Context: 2.5K + 0.6K = 3.1K tokens

You: "/draft"
   ↓
System loads: advertorial_master L2 (4K tokens)
Context: 2.5K + 4.6K = 7.1K tokens

Draft complete → You: "/gc" (garbage collect)
   ↓
System unloads L2, keeps L1
Context: Back to 3.1K tokens
```
